# JMES App

The JMES App is the official user client of JMES World and the official wallet for the JMES Blockchain.

Currently the App is released on Testnet and can be built from source on [Github](https://github.com/jmesworld/app/branches).

To join our Test Programme using Expo Go please contact the Mobile Team on their [Discord channel](https://discord.com/channels/.935880511950299217/1020254261768093716)
